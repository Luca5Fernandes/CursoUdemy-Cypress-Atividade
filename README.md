## :dart: Descrição

Projeto feito junto ao Curso Testes de aplicações modernas com Cypress Site Udemy, com o objetivo de desenvolver e aperfeiçoar várias 
técnicas de programação de Testes com o auxílio do Cypress.

- Visual Studio Code
- Cypress 
- TypeScript
- JavaScript
- Git e Git Hub
